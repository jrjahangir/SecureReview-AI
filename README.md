# SecureReview-AI

## Overview

SecureReview-AI is an AI-powered Application Security Assessment Platform.

The objective of this project is to combine Artificial Intelligence with traditional security analysis to identify security vulnerabilities in source code and generate professional security assessment reports.

---

## Objectives

- AI-assisted source code review
- OWASP Top 10 analysis
- CWE mapping
- Security recommendations
- PDF report generation
- Docker-based deployment

---

## Technology Stack

- Python
- FastAPI
- Docker
- Ollama
- Git
- GitHub
